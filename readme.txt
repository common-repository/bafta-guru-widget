== Description ==
The BAFTA Guru WordPress Plugin and Widget allows you to easily feature Guru Video, Audio and Article content on your own website.

Using the Plugin you can select the Craft categories to customise the video, audio and article content that is displayed on your website, via the handy BAFTA Guru Widget that is included as part of the Plugin.

== Installation ==

1. Upload the BAFTA Guru Widget plugin to your blog, Activate it, then Configure it (Settings > BAFTA Guru) to set your feed information and preferences.
2. Go to your blog Widget Manager (Appearance > Widgets).
3. Add the BAFTA Promo Widget to the appropriate sidebar, eg. Blog Sidebar, and set the widget preferences.
4. Click Save

Easy!
